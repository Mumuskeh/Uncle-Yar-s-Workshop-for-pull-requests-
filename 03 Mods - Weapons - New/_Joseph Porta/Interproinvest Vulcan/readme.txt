-Module:
Interproinvest Vulcan assault rifle [Joseph Porta]

----------------------------------------------------------
-Compatibility:
Note: You still need the original mod. Disable the original configs folder.

This module was done based on the DX8-9 package, but should be compatible with the Dx-11 files; in that case you may use all of the main files here except the files:
- mod_system_interproinvest_vulcan_base.ltx
- mod_system_wpn_interproinvest_vulcan_scopes01_Vanilla.ltx

----------------------------------------------------------
-Explanation:

NPC distribution. In addition to Army veterans, Army masters will carry it as well.
Trader distribution. In addition to Sidorovich, Army traders will sell it as well. It will be sold at one goodwill tier higher than before (from 3/5 to 4/5) by everyone.

Icon file trimmed.

Vanilla scopes, actor effects, parts, Grok's Enhaced Recoil, and Mags Redux files integrated. Yeah, the first three are offered by the installer as optional and "Gamma files", which is kinda dumb, because, you know, *VANILLA HAS ALL THAT AS A DEFAULT*. And who's gonna say no to being able to use the vanilla scopes? :V

Integration into:
Grok's Stash Overhaul
Lootboxes
PDA Interactive
Weapon Parts Overhaul

----------------------------------------------------------
-Patches:

Now these you can let them overwrite the DX11 files.

Vanilla silencer patch
To reduce item bloat, this will remove the included silencer. The gun will use the vanilla ATG silencer.

Sound performance patch
If you feel the audio causes a performance drop, try this patch that reduces the audo layers, and see if it helps.

BaS files
The the original configs file, but cleaned up a bit.
Do not use with the DX11 files.
