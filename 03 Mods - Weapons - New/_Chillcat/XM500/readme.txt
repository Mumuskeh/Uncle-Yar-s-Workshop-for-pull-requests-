-Module:
XM500 [Chillcat]

----------------------------------------------------------
-Compatibility:
NOTE: You still need the original mod. Disable the gamedata/configs folder.

If you use the .50 BMG and Barret M82 restoration module in this pack, the XM500 will use .50 BMG as proper instead of 12.7x50. This applies for Lewdnatic's .50 BMG pack module as well.
