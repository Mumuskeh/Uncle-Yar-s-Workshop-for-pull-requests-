- Grandpa Knife, by Schmittpm
Instead of replacing all vanilla knifes with new models, it introduces a new single one, "Grandpa's Old Knife", using the model and texture with the bear image.
You could technically use it with the replacements, but it's intended as a standalone.

Available in new game loadouts, Sid, and the day flea market.
Integrated into vanilla systems: corpse looting, new game loadout.
