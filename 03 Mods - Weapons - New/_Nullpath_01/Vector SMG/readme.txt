-Module:
Vector SMG [Nullpath_01]
Only available in the Gamma Discord.

----------------------------------------------------------
-Compatibility:
NOTE: You need the original mod. Disable the original configs and textures/ui folders.

Internal IDs have been reworked for better organization. Do not apply this patch to an existing save with the mod already in use, use a new game.

Once you apply the patch to the mod, do not mix with LEWD's Vector.

----------------------------------------------------------
-Explanation:

Raised value by 2k, and firerate to 1200 rpm as per the real gun.
Added default actor effects configs.
NPC loadout redistribution. Only masters from Freedom, ISG, Mercs, Monolith and loners will use it, with rare occurrences at lower ranks.
Trader redistribution. Freedom, ISG, Mercs and Monolith will sell it at medium-high goodwill ranks.
Icon file trimmed.

Integration into:
Grok's Enhanced Recoil
Grok's Stash Overhaul
Lootboxes
PDA Interactive
Weapon Parts Overhaul

-Patches:

Main files + 9x19 clone
Self explanatory. Same files, but with a 9x19mm chambered clone.
With Mags Redux, Glock 17 mags will be used.

Mags Redux
The weapon will use UMP45 magazines. Can be used with or without the 9x19 clone.
Use this or the AO Latecomers patch below, but not both (hence why it isn't integrated in the main files).

Arsenal OVerhaul Latecomers
For the basic package and with Mag Redux, the Glock 21 magazines will be used instead of the UMP-45 ones, as per the real life gun.
A Glock based .45 ACP drum will be added as well. No idea what other guns are out there that can use it other than this Vector clone and the Latecomers Glock 21, but uh, anyways.
For the .40 S&W module, another clone in such chambering will be added, using the Glock 35 mags of that module. Will also have its own drum mag.
