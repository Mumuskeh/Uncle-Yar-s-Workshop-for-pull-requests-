-Module:
P90 Tactical [Gunslinger]

This abomina-ahem, mod has been removed from Moddb, but in case you still have it or find it, you can use this.

----------------------------------------------------------
-Compatibility:
NOTE: You need to find the original gun, in a version of the Gunslinger files adapted to Anomaly that still has it.
Disable the folders items and loadouts in gamedata/configs, and the gamedata\meshes\anomaly_weapons\wpn_p90 folder.

----------------------------------------------------------
-Explanation:

This weapon inherited the data of the base P90. To ensure compatibility with other mods that rework the gun, the P90 Tactical has been independized away.
Text edited to have its own proper description and a short name.
Scripted NPC loadouts files converted to DLTX.
Trader distribution. To accommodate it in the higher supply tier, the normal P90 is moved down one supply lower.
Added default actor effects settings.

Integration into:
Lootboxes
Grok's Enhanced Recoil
Grok's Stash Overhaul
PDA Interactive
Weapon Parts Overhaul

----------------------------------------------------------
-Patches:

Boomsticks and Sharpsticks
Cleaned up files to use the BaS scopes.

Mags Redux Standalone
If you don't want the weapon (say because you don't want bloat in your game), this will add the extended P90 mag in the mod and nothing more.
They're not assigned to guns carried by NPCs, and must be bought.
