-Module:
Scorpion EVO 3 [Dionisiy9x39]

----------------------------------------------------------
-Compatibility:
NOTE: The original mod is still needed. Disable the gamedata/configs folder.

----------------------------------------------------------
-Explanation:

NPC loadouts redistributed. Now the Army and Duty will also carry it, in addition to bandits, renegades, Freedom, and Mercs. Veterans will carry a default configuration, and masters a randomized. Only masters will carry it among loners.
Icon file trimmed. Seems like some repainted variants in gray and desert camo were intended (the textures are even there for that), but as it is now, those icons merely bloated the file.

Integration into:
Grok's Stashes Overhaul
Lootboxes
PDA Interactive
Weapon Parts Overhaul

----------------------------------------------------------
-Patches:

Model tweaks
The models for the 5C Kit upgrade will now have the laser module attached at the right side of the rifle, instead of the left (?).
The models for the BaS Trijicon and Spectre combo packs with side red dot sights have the laser module removed (I mean, BaS itself didn't add laser modules with those, why should this gun?).

Boomsticks and Sharpsticks ammo patch
Clones the gun into the 9x21mm variant. It will use BaS' 9x21 ammo, silencer, and mags. It will be only be carried and traded by Duty and the Army.
Integrated support for Armor Modkits addon allows you to use modkits to change the chambering of a given gun.

Armor Modkits addon
This is intended for BaS Lite users. This weapon can use the Special AK 5C Kit from BaS, but BaS Lite offers options to remove the upgrade kits. This patch allows you to use a modkit for the upgrade.
