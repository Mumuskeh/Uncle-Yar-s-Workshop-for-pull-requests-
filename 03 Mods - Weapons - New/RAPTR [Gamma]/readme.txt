-Module:
RAPTR Shotgun [GAMMA]

----------------------------------------------------------
-Compatibility:
This gun is only available in GAMMA. My patches are usually oriented to vanilla Anomaly, but you may use it all the same. Or if it ever becomes available in vanilla (pester the Gamma folk for it).

----------------------------------------------------------
-Explanation:

The icon file has been simplified and reduced through the use of icon layers.

Mags Redux support: the files for the unique RAPTR drum mag has been removed/disabled, and the gun will use the USAS12 one. They're basically the same, just with different models. So it seemed silly and redundant.

Lootboxes
PDA Interactive
Weapon Parts Overhaul
