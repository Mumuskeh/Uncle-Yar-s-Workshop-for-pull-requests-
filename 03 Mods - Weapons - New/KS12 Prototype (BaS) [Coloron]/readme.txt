-Module:
KS12 Prototype [Coloron]
Only available in the Gamma discord.

----------------------------------------------------------
-Compatibility:
NOTE: You need the original mod. Disable the original configs and scripts folders.

This is an extension for a Boomsticks and Sharpsticks gun, so BaS is a requirement.
If you use BaS Lite, you need the Saiga module.

----------------------------------------------------------
-Explanation:

Adapts the mod for vanilla use. It can be used either with BaS alone or BaS Lite.

The text will now identify the new gun with the much more explicit "KS-12 Komrad ISG (Custom)" ("KS12 Prototype"? Prototype of what?), and the Saiga-12 ISG as "KS-12 Komrad ISG".
The description of both guns now state the possiblity of attaching a grenade launcher after upgrades.

As vanilla lacks access to a Gamma mod where Nimble makes weapon upgrades, we will instead have him sells the new gun at a pretty penny.

Files consolidated.

Added default actor effects configs.

Integration into:
Armor Modkits addon. Instead of buying it off Nimble, you can instead upgrade a Saiga12 ISG with a custom gun modkit.
Grok's Enhanced Recoil
Grok's Stash Overhaul
PDA Interactive
Weapon Parts Overhaul
