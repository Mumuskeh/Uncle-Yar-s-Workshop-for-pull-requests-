-Module:
StG44 [LEWD]

----------------------------------------------------------
-Compatibility:
NOTE: You need the original mod. Disable the original configs folder.

This module requires a fresh game due to renamed internal IDs. If you were using the mod in a game, you must start a new one.

----------------------------------------------------------
-Explanation:

Ammo file fixed. The data for the 7.92x33 Kurzpatrone 43 m.E. round was being used for the 8x33 FNM FMJ BT in bad status, instead of being its own entity.
Text files consolidated. Edited the english description to be a bit shorter for the guys playing in small resolutions.
NPC loadouts. Moved file to proper directory. Certain factions (Army, Duty, Freedom, Ecologists, ISG, mercs) won't carry it anymore.
Trader redistribution. Removed from Sakharov. Fixed Flea Market files.
Added files for the Cordon Army trader, ISG extra traders, and some of the basic trade configurations (the ones used in Warfare after a faction's trader is dead, I think).

Mags Redux support. It didn't cover all ammo it should. The mag is also removed from traders (Mag redux itself will distribute it).

Integration into:
Grok's Enhanced Recoil
Grok's Stash Overhaul
Lootboxes
PDA Interactive
Weapon Parts Overhaul
