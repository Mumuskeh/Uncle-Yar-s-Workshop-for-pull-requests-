-Module:
Shak-12 Tactical [LEWD]

----------------------------------------------------------
-Compatibility:
NOTE: You need the original mod. Disable the configs folder

Important: If you were already using the mod in a game, you'll need to start a new one.

----------------------------------------------------------
-Explanation:

Internal IDs renamed, for better sorting in the inventory along Blackgrowl's AShey 'n ShAKky.
Text edited. "(LEWD)" tags are removed.
The original mod only added it to mercenaries only. Now it does for some of the same traders as Ashey 'n Shakey does for its Shak-12: Army, Duty, Mercs, Monolith, Owl, and giving it to master/legend members of only those factions. Though at lower odds of being in stock/spawning with it than Ashey 'n Shakey's Shak-12.
Icon file trimmed. Repeated icons for each scope have been removed, and instead overlays are now used.

Integration into:
Grok's Stash Overhaul
Lootboxes
PDA Interactive
Weapon Parts Overhaul

----------------------------------------------------------
-Patches:

Armor Modkits addon
You may upgrade Blackgrowl's Ashey 'n Shakey' Shak-12 to Lewdnatic's Shak-12 Tactical, and this to the Operator and Specialist variants without the original kit addon.
Do not use the patch included in the Armor Modkits addon installer, it is intended for the unmmoded Shak-12 Tactical.
