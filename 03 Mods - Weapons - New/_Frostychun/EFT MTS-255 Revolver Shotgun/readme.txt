-Module:
EFT MTS-255 Revolver Shotgun [FrostyChun]

----------------------------------------------------------
-Compatibility:
NOTE: You need the original mod. Let this overwrite it.

IMPORTANT: A new game is required if you were already using the mod.

----------------------------------------------------------
-Explanation:

Internal IDs renamed, because my nascent OCD was screaming in my ear. And should make it easier to find in inventory near the MP shotguns.
NPC distribution. Given to experienced stalkers, with the number of entries reduced to one for each. The mod tries to give the gun to novices *and* trainees. In vanilla Anomaly, except for Sin trainee rank stalkers have no loadouts, instead they inherit the loadout of novices. Which is mostly useless. That, and making it a novice tier gun seems weird when it looks to be a little better than your TOZes and it's not sold anymore (then again, alternative history).
Trader distribution. It will be sold by factions whose NPCs carry it at a low-mid goodwill tier.
Icon file trimmed.

The world model has been fixed, and won't crash the game anymore.

Grok's Stash Overhaul
Lootboxes

Boomsticks and Sharpsticks

Revolver Parts
Reworks the patch already included in that mod to acknowledge the changed ID.
Includes option for the BaS patch above, though there are no 20x70 cylinder, so the 12 one is used for now.
