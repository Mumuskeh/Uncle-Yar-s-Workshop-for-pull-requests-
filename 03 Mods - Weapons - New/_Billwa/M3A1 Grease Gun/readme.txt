M3A1 Grease Gun [Billwa]

----------------------------------------------------------
-Compatibility:
NOTE: You still need the original mod. Disable the original configs folder.

Due to changed internal IDs, if you were using the mod in a save already, you'll need to start a new game with this patch.

-Explanation:

Internal IDs renamed, so both guns are ordered together in the inventory.
Files consolidated.
The tactical version gets its value and weight increased.
Trader redistribution. The tactical version won't be sold at the lowest goodwill tier (what's the point of the basic version then anyway?). It will be offered two goodwill tiers above now.
Audio files commented for NPC reaction.
Settings for actor effects and Grok's Enhanced Recoil tweaked to follow the UMP45, not the MP5 (the M3A1 uses .45 ACP, not 9mm).
Icons gathered into a single file.
