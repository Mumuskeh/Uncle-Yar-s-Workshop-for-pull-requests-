-Module:
Colt M733 and XM177 Carbines [Jmerc]

----------------------------------------------------------
-Compatibility:

NOTE: You still need the original mod. Disable the original configs folder

Important: This patch requires a new game if you were already using this mod.

----------------------------------------------------------
-Explanation:

Guns separated. They're very similar, so this way you can choose to opt out of one or the other.
Identified as "Colt CAR-15 Commando XM177" and "Colt Model M733 Commando", with the number models used as short names.
Internal IDs renamed, so that both weapons, both technically variants of the M16, are sorted together in inventory and along other M16 rifles.
Similarly, upgrade scheme changed from the one used for the G36, to the one used for the M16.
Icon file trimmed.

Integrated into:
Grok's Stash Overhaul
Lootboxes
PDA Interactive
Weapon Parts OVerhaul
