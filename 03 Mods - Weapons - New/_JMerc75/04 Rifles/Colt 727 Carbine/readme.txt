NOTE: You still need the original mod. Disable the original configs folder

-Explanation:

Identified now as "M16A2 Carbine Model 727", with "M16A2 727" as short name.
Internal ID renamed, as the weapon, technically a variant of the M16A2, are sorted together in inventory and along other M16A2 carbines (or M16 rifles).
Similarly, upgrade scheme changed from the one used for the G36, to the one used for the M16.
NPC loadouts. The mod assigns the gun to one or two of each faction's ranks. I make it so only the highest one of the two ranks may have random ammo loaded.
Traders. Added to mercenaries. I was going to remove it from from Duty and Army, but it's such a simple AR rifle, that, eh, whatever.

Icon file trimmed.
