-Module:
M1 Garand Tactical REDUX [JMerc] 
A Gamma mod, but available in their Discord server.

----------------------------------------------------------
-Compatibility:
NOTE: You still need the original mod. Disable the original gamedata/configs folder.

----------------------------------------------------------
-Explanation:

Identified as the US Navy's "M1 Garand Mod 1", as the gun in this mod is chambered in 7.62 like such model, and the original Garand was chambered in .30-06 Springfield.

Icon file trimmed.

Mag Redux support split off as a separate patch, as 1) the files don't seem to be working at least in vanilla, and 2) Mags Redux is not actually designed for clips. It also facilitates making the clip compatible for JMerc's standard M1 Garand mod (.

-Patches:

Neko's 30-06 Springfield
Adds a version of the gun in the original .30-06 Springfield chambering.
With Armor Modkits addon, the weapons can be rechambered back and forth.

Mag Redux
[NOT WORKING, I can't get this to work on vanilla Anomaly. Offered as reference]
Split off the main files, reworked to be compatible with JMerc's standard M1 Garand mod, and added option for .30-06 Springfield.

Armor Modkits addon + M1 Garand
Allows to upgrade Jmerc's basic M1 Garand to the Tactical one. Includes option for .30-06 Springfield.
While the main file is identical to the patch in Armor Modkits addon's installer, the .30-06 option is not compatible with it. Do not mix.
