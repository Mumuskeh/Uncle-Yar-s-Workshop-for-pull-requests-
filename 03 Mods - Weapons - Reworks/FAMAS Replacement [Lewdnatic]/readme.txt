-Module:
FAMAS Replacement [LEWD]

----------------------------------------------------------
-Compatibility:
NOTE: You need the original mod. Disable the original configs folder.

----------------------------------------------------------
-Explanation:

Disabled the laser configuration for the basic F1 and G2 (the models don't have a LAM anyway).
Text edited to remove any mentions of "LEWD".

Trader files. The original mod offered all guns in traders as soon as you reach the second goodwill level. So I tiered all guns for some pregression.
Mercs and ISG get all.
Freedom won't get the Commando.
The Army ignores the Felin set and the Commando.
The flea market will only sell the basic F1 and G2 rifles.
Spirit is a barman with only one supply level, dunno why the mods gives him anything and in the wrong level... but I've fixed it, though only for the F1.
Disabled the trade file for Duty, as it doesn't fit their theme (the Army still has a pass, WP weaponry or not, if only to have samples).

You can drag the Felin scope to the F1 Felin to attach it, or drag a swiss knife or multitool on it to separate them (you will only get the unscoped F1 Felin).

-Patches:

Lite
Disables the Felin set and the Commando (which gathered dust in a warehouse because it was so meh nobody bought it off France. Given it's more a crappy SMG than a real rifle, maybe we can make a 9mm variant like the AUG but Zone made?).

Lite+
Also disables the Tactical variants, leaving only the basic F1 and G2.

Mags Redux
Adds a set of mags for the F1 rifles: a standard 25 rounds FAMAS magazine, with icon and world model, and converted fast, extended, and casket STANAG magazines (as in, they're already converted and can't be retooled. The logic being that converting them for the FAMAS system is a bit too destructive).
I've already submitted a pull request with this addition to the main Mags Redux github repo, so that even the vanilla FAMAS can benefit of it, but it may take time to be applied. So the patch is offered in two ways:
-Standalone: It includes everything needed for the patch. Use this with the current 2024 update.
-Updated: it only includes the mag-to-gun assignments and the world model. Use this if Mag Redux accepts the pull. Or if you use my fork (see the Mags Redux module readme in this package), which includes it.
