-Module:
XM8 Reanimation and Model [tint08771]

----------------------------------------------------------
-Compatibility:
NOTE: You still need the original mod. Disable the gamedata/configs folder.

The XM8 Reenabler from DLTX Minimod is not needed anymore. Disable it if you were using it.

----------------------------------------------------------
-Explanation:

Fixed stats to that of the vanilla XM8, including push power, damage, muzzle velocity, recoil, degradation, hiding crosshair on aiming,and weight with scopes. Seems like the reanimation was using stats from a version older than 1.5.3?

NPC loadouts and trader distribution, so you don't need a XM8 reenabler.
Masters of Freedom, ISG, Mercs, and Monolith have 50/50 chances of carrying one. The same factions will sell it at high goodwill tier, plus Nimble.
