-Explanation:

Several internal IDs renamed for better inventory ordering with equivalent outfits.
Fixed Monolith trader not selling its outfit.
Removed references to non existent outfits, which may cause crashes.
Fixed outfit types: Grayness outfit turned from scientific to medium, Veto from light to medium, Sky Dome from medium to exo.
The Sky Dome is now based on Kronq's Exosevas.
Thermal overalls are now a variant of the hybrid.
NPC spawn files now should avoid conflicts.
Trader redistribution. Some traders have no business selling outfits from other factions, so I take them away.
New game loadouts. Removed changes unrelated to adding the outfits, with the exception of the vanilla Berill outfit becoming a purchasable option.
Added parts to outfits.
Added default actor effects settings.
English translation edited. Most of it should now flow better and follow the vanilla text for whatever applies. Not completely done and I took some liberties, but should read better than before.
Icon file trimmed, both in size and unused icons. Adjusted the positions of some icons to proper form.
