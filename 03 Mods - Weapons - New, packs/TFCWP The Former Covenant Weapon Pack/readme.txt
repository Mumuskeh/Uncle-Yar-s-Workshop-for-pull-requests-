-Explanation:

Consolidated files and icons.
Reenabled shell ejection effects to the Vz61 reanimation.
Trader distribution. High goodwill will usually be required.

Moved the HUD FOV 0.4 file to a different folder as optional.

Integration into:
Grok's Enhanced Recoil
Grok's Stash Overhaul
Lootboxes
PDA Interactive
Weapon Parts Overhaul


Two options for both GTorquemada's basic package in Moddb, and ATHI's expanded edit in Github.

For GTorquemada's basic package in Moddb:
- A Lite option is included where only the RPK-16 and the Vz.61 reanimation will be enabled in game. This is due to the MK47 and MP155 models having a dumb number of materials, making them harder for potatoes, and honestly, being just sligtly better AR rifle and MP153. The RPK16 is just the BaS one with a drum mag, but it doesn't have a dumb number of materials.
- A "Vz.61 Reanim Only" option further reduces this to only the Vz.61 reanimation. In case that's the only thing you're interested, be it because with Mags Redux the drummed RPK16 is redundant, or you don't want more weapons.

A WIP option for ATHI's expanded edit in Github includes the same patching as GTorquemada's, but the 3DSS configs (that I could see) moved to a separate file. This merely opens up the possibility of using scopes for DX8-9 players (not collimators, you'll get a shader crash), although for now only the ACOG has a scope texture assigned. This is not recommended for use yet.
