-Module:
RWAP [Retrogue]

----------------------------------------------------------
-Compatibility:
NOTE: You need the original mod.
If you use the consolidated files, disable the gamedata/configs folder.

----------------------------------------------------------
-Explanation:
Icon files have been trimmed.

Fixed up some descriptions. For example, some were a jumble, or unedited. And the HK G95K didn't have a description, so now it has one from Wikipedia (boring, but serviceable).

An option for Consolidated Files also trims the number of files, leave only one for a given purpose (one loadout file, one parts file, etc).
This is intended for full RWAP installations. I haven't tested it, but if you didn't select a single weapon, the game may crashes.


Consolidated files also include:

Distribution to NPCs done more sparingly, instead of everyone getting everything with AP ammo. Some factions will carry more guns than others. Veterans carry some guns in default configurations and others in completely random ones, while masters carry all guns in completely random configurations. Masters will carry any tactical variants when available, while veterans carry the basic varants (exceptions aside).
Distribution to traders, in similar fashion to NPCs above, and allow you to sell the weapons.
Default actor effects settings.

Integration into:
Grok's Enhanced Recoil
Grok's Stash Overhaul
Lootboxes
PDA Interactive
Weapon Parts Overhaul
