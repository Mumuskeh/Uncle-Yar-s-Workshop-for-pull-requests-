-Module:
RWAP [Retrogue]

----------------------------------------------------------
-Compatibility:
NOTE: You need the original mod.

If you use the consolidated files or the group modules, disable the gamedata/configs folder, and start a new game if you were already using RWAP in your current one.

----------------------------------------------------------
-Explanation:

-Standard files:
Icon files have been trimmed.
Fixed up some descriptions. For example, some were a jumble, or unedited. And the HK G95K didn't have a description, so now it has one from Wikipedia (boring, but serviceable).
Use these if you picked only some guns during installation. You only need to go into gamedata\configs\items\weapons and remove the files for those you didn't install.

If you want 0 changes with the most compatibility with other mods and patches, stay with these.


Some options options are offered, with further changes.

-Consolidated files:
Use these if you want all guns in RWAP (you still need the whole of its assets). A new save is needed if you were already using RWAP in your current save.
All files are consolidated, leaving only one for a given purpose (one loadout file, one parts file, etc).
Distribution to NPCs done more sparingly, instead of everyone getting everything with AP ammo. Some factions will carry more guns than others. Veterans carry some guns in default configurations and others in completely random ones, while masters carry all guns in completely random configurations. Masters will carry any tactical variants when available, while veterans carry the basic variants (exceptions aside).
Distribution to traders, in similar fashion to NPCs above.
Default actor effects settings.

The AKMSU and MCX weapons had their internal IDs renamed, so they may be ordered in the inventory better and together, even with MCX guns from other mods.

Integration into:
Grok's Enhanced Recoil
Grok's Stash Overhaul
Lootboxes
PDA Interactive
Weapon Parts Overhaul


-Group modules:
These include all the changes in the consolidated files above.
Weapons will are grouped up in some ways, so that mix and matching with other mods that include the same guns is easier, without going through each gun one by one during installation-. Think of it like BaS Lite.

--Scopes
--HK416
--Sig 553
--SIG MCX
	This includes the Virtus, Rattler, Spear-LT, and MPX
--Modified AKs
	This includes the SAG-AKMSU, AM-17, AMB-17, and MB-47.
--Modified M4
	This includes the BCM CQB MCMR-10, KS-1 L403A1, L119A2, Noveske Gen4 Shorty, and M4A1 URG-I.
--Snipers
	This includes the Crom CR-7 and KAC M110K1.
--Others
	This includes the HK G95K and TTI JW4 Dracarys Gen-12

-Patches:

Honey Badger & .300 Blackout Framework [Pillii]
This requires both Pillii's mod, and the patch for it in the "02 Mods - Ammo" folder in this package.
Several weapons now have variants in .300 Blackout. Two options:
- SIG MCX: variants for the MCX Virtus, MCX Rattler, and MCX SPEAR LT.
- SIG 553: variants for the SG-553R SBR, and SG-553R SBR Tactical.
The two 553 will use the SG mag in the Extra Mags patch above.
