-Module:
Make the L85A1 trash [Hisalute]

----------------------------------------------------------
-Compatibility:
The original mod is not needed.

----------------------------------------------------------
-Explanation:

DLTX conversion of both options.
Do note that due to how DLTX works, the A2 would inherit the changes too. So I restate its vanilla values and make the file load pretty late. This means if you have something else trying to modify the same values in the A2, they'd become void.

JMerc's L86 LSW will also be covered.
