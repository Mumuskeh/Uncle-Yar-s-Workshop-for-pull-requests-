-Module:
Shell Particles Redone [SoulCrystal]

----------------------------------------------------------
-Compatibility:

NOTE: You still need the original mod, and you risk crashing otherwise.
Make sure you install all options for this to work.

----------------------------------------------------------
-Explanation:

Auxiliary patch to cover mod weapons that could use the new shell particles.

Currently, it mostly covers pistols, and weapons chambered in 7.62x25, 5.7x28, .50 BMG, .50 AE... from:
-Boomsticks and Sharpsticks
-Simple Weapon Pack
-Misc. Weapon Pack
-RWAP
-Several loose mods (Jmerc's, Lewdnatic's...)

It seems there was an intention to create more particle effects where you could assign the shell flying in different directions, but it's either incomplete, or SoulCrystal didn't use it much on the original mod.
