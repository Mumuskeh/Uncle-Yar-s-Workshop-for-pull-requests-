-Module:
Ammo Pack .50 BMG [LEWD]

----------------------------------------------------------
-Compatibility:
Note: The original mod is still required. Disable the configs folder.

If you were already using the mod in a game, you'll need to start a new game.

----------------------------------------------------------
-Explanation:

Internal IDs renamed, to follow the vanilla convention. This helps the new ammo blend with the vanilla ammo in the inventory better, instead of making it stick out apart. Helpful if you've restored the vanilla .50 BMG so the entire ammo type can be together.

Integration into the vanilla stash system.

Integrated into:
Arszi's Mutant Bleeding + Arszi's Mutant Bleeding Optimization Pach
Arti's Ballistics. This is a **VERY SIMPLISTIC** integration and won't go far. Bug the author for proper support.
Grok's Stash Overhaul
Lootboxes
PDA Interactive

----------------------------------------------------------
-Patches:

DX8-9 patch
This gathers the non hq icons into the same file and makes it DX8-9 compliant.
Offered separately for compatibility reasons.
Do not install the mod's original patch.

.50 BMG and Barret M82 restoration patch
The mod straight up changes the ammo selection of the M82, Lewdnatic's wacko DEagles, Sedzhimol's Gepard 6 Lynx, and Chillcat's XM500.
This will also add to the selection the vanilla .50 BMG, so you can use it as a default/cheap option, and integrates the new ammo to the Mags Redux changes.
It requires the .50 BMG and Barret M82 restoration in this pack.
