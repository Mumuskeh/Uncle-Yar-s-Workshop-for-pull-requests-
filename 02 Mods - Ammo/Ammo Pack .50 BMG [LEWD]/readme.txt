DX8-9 patch
This gathers the non hq icons into the same file and makes it DX8-9 compliant.
Offered separately for compatibility reasons. Do not install the mod's optional patch.

.50 BMG and Barret M82 restoration patch
The mod straight up changes the ammo selection of the M82, Lewdnatic's wacko DEagles and some "gm6" for its ammo. The XM500 module has its own patch.
This will also add to the selection the vanilla .50 BMG, so you can use it as a default/cheap option, and integrates the new ammo to the Mags Redux changes.
It requires the .50 BMG and Barret M82 restoration in this pack.
