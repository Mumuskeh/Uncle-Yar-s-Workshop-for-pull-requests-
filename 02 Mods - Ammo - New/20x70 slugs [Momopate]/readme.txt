-Module:
20x70 slugs [Momopate]

----------------------------------------------------------
-Compatibility:
The original mod isn't needed. It does however require Boomsticks and Sharpsticks.
If you use BaS Lite, you'll need the 20x70 ammo module.

If you use a mod that includes an older version of the addon, such as ATHI's Taurus Judge 20, or the Revolver's Revolver Pack from the Gamma Discord server, Disable this file from the configs folder:
	mod_system_judge_slugs_compat.lt
And every file ending including 20x70 in its filename.

----------------------------------------------------------
-Explanation:
Standalone version of the original mod that can be used in vanilla, with the author's permission.

It adds two slugs variants of the 20x70 gauge, the Devastator and Star AP, plus bad and damaged versions for those and the standard BaS 20x70 buckshot.

This version fixes and cleans things a little, and ensures the older ammo variants are hazardous to weapon degradation.
Icons are gathered into a single file and made DX8-9 compliant.

Several mods are recognized to load the ammo (most were already there, others I added):
-Misc. Weapon Pack's Taurus Judge
-ATHI's Taurus Judge
-Boomsticks and Sharpsticks
-BaS Extra Weapons
-Speedloaders and ammo for revolvers

Integration into:
Grok's Stash Overhaul
Lootboxes
PDA Interactive
